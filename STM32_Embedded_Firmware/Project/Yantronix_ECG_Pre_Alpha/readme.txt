To create Image header files using the tool "bmp2glcd.exe":

1)Gimp->Image->Scale Image
2)Gimp->Image->Mode->Grayscale
3)Gimp->Image->Flatten Image (To remove transparency)
4)Gimp->Colors->Auto->Equalize or Stretch Contrast (depending on quality)
5)Gimp->Save->Quit
6)Paint->File->Open Image file
7)Paint->Save as type->Monochrome Bitmap
8)Paint->Image->Flip Vertical
9)Paint->Save->Exit
10)bmp2glcd->Image file
